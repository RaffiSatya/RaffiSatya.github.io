Minggu 1 dan 2: HTML

HTML adalah singkatan dari Hypertext Markup Language, yaitu bahasa markup standar untuk membuat dan menyusun halaman dan aplikasi web. dengan HTML, kita dapat membuat konten website sederhana yang berisi tulisan, gambar, tabel, dan lain sebagainya

Minggu 3: CSS dan Javascript

CSS merupakan bahasa yang digunakan untuk mengatur tampilan elemen yang tertulis dalam bahasa markup, seperti HTML. CSS berfungsi untuk memisahkan konten dari tampilan visualnya di situs. Sedangkan Javascript adalah bahasa pemrograman yang digunakan untuk membuat halaman web yang interaktif. Dengan Javascript, dapat dibuat sebuah fungsi dan program yang ingin digunakan.

Minggu 4: Bootsrap

Bootstrapmerupakan sebuah framework javascript yang sering digunakan untuk membuat web. Pada minggu ini, dipelajari cara membuat web menggunakan framework bootstrap serta mengisinya dan mengatur stylingnya

Minggu 5 dan 6: Leaflet JS

Leaflet JS merupakan sebuah WebMap API yang berguna untuk menambahkan peta pada sebuah web. Pada minggu ini, diajarkan cara menambahkan peta dan juga GeoJSON ke dalam web serta melakukan layouting dan menambahkan plugin-plugin bootstrap
